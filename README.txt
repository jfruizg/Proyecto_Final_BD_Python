Progama basado en FLASK Y SQLALCHEMY concetado a una base de datos en SQL

Instalacion y configuracion del programa

- Primero revisar si se tiene instalado pip
    python --version

- Instalacion del flask
    pip install flask

-Instalar SQLAlchemy
    pip install Flask-SQLAlchemy

- Instalar en modulo 'MySQLdb
    pip install mysqlclient

- Instalar Catchap
    pip install -U Flask flask-session-captcha Flask-Sessionstore pymongo
    pip install Flask-reCaptcha

- Ingresar al entorno virtual en windows
    ./venv/Scripts/activate

GITHUB COMANDOS

    -git branch // Saber la rama la cual estas ubicado y las demas ramas local

    crear rama

    -git branch Readme_Git

    -git add . // archivos adicionales

    -git commit -m "" // commit , hacer los cambios en la rama o master - tener cuidado donde estas ubicado

    -git push --set-upstream origin Readme_Git // cuando vas hacer el primer git push a la rama
