Progama basado en FLASK Y SQLALCHEMY concetado a una base de datos en SQL

Instalacion y configuracion del programa

- Primero revisar si se tiene instalado pip
    python --version

- Instalacion del flask
    pip install flask

-Instalar SQLAlchemy
    pip install Flask-SQLAlchemy

- Instalar en modulo 'MySQLdb
    pip install mysqlclient

- Instalar Catchap
    pip install -U Flask flask-session-captcha Flask-Sessionstore pymongo
    pip install Flask-reCaptcha
